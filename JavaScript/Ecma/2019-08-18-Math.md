## Math属性和方法

Math 是一个内置对象， 提供数学常数和函数的属性和方法。

### Math.PI

圆周率，一个圆的周长和直径之比，约等于 3.141592653589793。

```javascript
Math.PI // 3.141592653589793
```

### Math.abs()

获取绝对值。

```javascript
Math.abs(-1)  // 1
Math.abs(-true) // 1
Math.abs('1') // 1
```

### Math.ceil()

向上取整。

```javascript
Math.ceil(12.1) // 13
Math.ceil(-12.9) // -12
Math.ceil('12.1') // 13
```

### Math.floor()

向下取整。

```javascript
Math.floor(12.6) // 12
Math.floor(-12.1) // -13
Math.floor('-12.1') // -13
```

### Math.round()

四舍五入。

```javascript
Math.round(12.5) // 13
Math.round(-12.4) // 12
Math.round('12.5') // 13
```

### Math.max()

取一组数的最大值。

```javascript
Math.max(12, 23, 13, 24, 23, 24, 25, '34', 15); // 34
```

### Math.min()

取一组数的最小值。

```javascript
Math.min('12', 23, 13, 24, 23, 24, 25, 34, 15); // 12
```

### Math.random()

获取[0-1]之间的随机小数。

```javascript
Math.random() // 0.8184921957461317
Math.random() // 0.351685690699056
Math.random() // 0.4972569148842041
```

### Math.trunc()

去除一个数的小数部分，返回整数。

```javascript
Math.trunc(1.23456) // 1
Math.trunc(1.23456) // 1
Math.trunc('1.23456') // 1
```

### Math.sign()

判断一个数是正数(1)、负数(-1)、还是零(0)。

```javascript
Math.sign(6) // 1
Math.sign(-6) // -1
Math.sign('6') // 1
```

### Math.imul()

两个数相乘。

```javascript
Math.imul(2, 3) // 6
Math.imul('2', '3') // 6
```

## 备注

Math方法都会调用 `Number()` 进行转换，遇到非数字会默认调用 `Number()` 进行转换。
